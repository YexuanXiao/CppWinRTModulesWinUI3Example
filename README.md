The initial commit of the repository is the project template of WinUI3, and the second commit changes it to use C++/WinRT 3.0 modules.

Note that I prefer not to use pch and import headers.

I have provided the NuGet package [YexuanXiaoCppWinRTPlus.Ext](https://www.nuget.org/packages/YexuanXiao.CppWinRTPlus.Ext), which offers modules to cover the additional header files provided by the Windows App SDK.
